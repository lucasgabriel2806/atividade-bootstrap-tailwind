Curso HTML, CSS & JavaScript Novo

HTML é uma linguagem de marcação, usada para dar estrutura para a página web através de elementos / tags, exemplo: <html>. Se quisermos colocar texto, imagem, vídeo, lista, link, etc., usamos tags HTML epecificas para adicionar eles ao site, pois tudo isso é o que compoem a estrutura do site. Todo site usa HTML, mas não pode ter somente estrutura, visualmente ele não é atrativo, precisamos adicionar estilo.

CSS é uma linguagem de estilização usada em conjunto com o HTML para estilizar uma página web. Se eu quiser mudar a cor de um texto, mudar a cor de um backgroun, mudar a fonte de um texto, diminuir ou aumentar o tamanho de um container, um elemento ou imagem, nós podemos usar CSS pra estilizar tudo isso. CSS trabalha com estilização através de classes e ids que atribuimos a elementos HTML, exemplo:.texto-azulum bom site precisa de um bom conteúdo e um bom estilo, mas ainda falta a interação.

JavaScript é uma linguagem de programação para web usada para dar iteratividade em um site através de códigos que acessam elementos e adicionam funcionalidades a eles, também é possível mudar estilos através do JavaScript. um exemplo de código JavaScript: window.alert("Bem vindo!");. Existem diversas bibliotecas e frameworks para JavaScript, como React, vue.js, JQuery, entre outros.
